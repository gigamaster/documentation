    Welcome to the Item API! You can use our API to access Item API endpoints, which can get information on various items of data in our database.

    We have language bindings in Shell, PHP, and JavaScript! You can view code examples and you can switch the programming language of the examples with the tabs in the top right.

    This example API documentation page was created with to test. Feel free to edit it and use it as a base for your own API's documentation.

    Authentication

    To authorize, use this code:

    # With shell, you can just pass the correct header with each request
curl "api_endpoint_here"
  -H "Authorization: userapi"

    const item = require('item');

let api = item.authorize('userapi');

    Make sure to replace userapi with your API key.

    Item uses API keys to allow access to the API. You can register a new Item API key at our developer portal.

    Item expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: userapi

    You must replace userapi with your personal API key.

    Items

    Get All Items

    curl "http://example.com/api/items"
  -H "Authorization: userapi"

    const item = require('item');

let api = item.authorize('userapi');
let items = api.items.get();

    This endpoint retrieves all items.

    HTTP Request

    GET http://example.com/api/item

    Query Parameters

    Parameter Default Description
    include_files false If set to true, the result will also include files.
    available true If set to false, the result will include files that have already been edited.
    Remember — an item is a real data!
