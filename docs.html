    Prerequisites

    The user-friendly solution is to download :

    • Virtualbox
    • Vagrant

    Then, clone the repo to your local machine and start with command in a terminal 'vagrant up'

    It will download, installl and setup a virtual machine.

    Locally

    You're going to need:

    • Linux, macOS or Windows
    • Ruby, version 2.3.1 or newer
    • Bundler â€” If the bundle command doesn't work, just run gem install bundler in a terminal.

    Getting Set Up

    1. Fork this repository on GitHub.
    2. Clone your forked repository (not our original one) to your hard drive with git clone https://github.com/YOURUSERNAME/documentation.git
    3. cd slate
    4. Initialize and start. You can either do this locally, or with Vagrant:
    # either run this to run locally
bundle install
bundle exec middleman server

# OR run with vagrant
vagrant up

    You can now see the docs at http://localhost:4574.

    Now that all is set up on your machine, you can start editing with HTML and Markdown.
